## AI Tools Used

I used copilot with the Claude Sonnet 4.6 model.

## AI Usage

I used AI to create a basic web API project in C#. After creating the project I asked the AI to create test cases for a specific function. When I was happy with the test cases I asked the AI to write the function. I repeated this process until I was happy with the project.

## Examples Of Where AI Helped

AI greatly increased the speed at which I was able to complete this project. It also thought of test cases that I didn't initially think of.

## Examples Of Where I Corrected The AI

I changed the endpoint names to make more sense than what the AI originally generated. When the AI initially created the project it used dotnet 7. I asked it to switch the project to dotnet 8 because 7 is no longer supported.

## Code Or Tests Written Without AI

None. I thought it would be faster to have the AI write everything.
